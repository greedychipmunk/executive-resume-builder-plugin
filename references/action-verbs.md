# Action Verbs Reference

Strong resume bullets begin with a precise, past-tense action verb (present tense
only for the current role). The verb should signal *what kind of impact* the
candidate had. Avoid repeating the same verb; avoid weak openers entirely.

## Choose verbs by the kind of impact

### Led / Owned (scope, authority)
led, directed, headed, oversaw, managed, supervised, chaired, governed, owned,
orchestrated, spearheaded, piloted, presided, coordinated, marshaled

### Built / Created (you made something new)
built, created, designed, architected, founded, established, launched, developed,
engineered, formulated, devised, instituted, pioneered, originated, introduced

### Grew / Increased (positive quantitative change)
grew, increased, expanded, scaled, accelerated, boosted, drove, generated,
amplified, maximized, doubled, tripled, multiplied, surpassed, outpaced

### Reduced / Saved (cost, time, risk down)
reduced, cut, decreased, eliminated, streamlined, consolidated, minimized,
saved, trimmed, lowered, slashed, de-risked, automated

### Improved / Transformed (qualitative change)
improved, transformed, overhauled, modernized, revamped, restructured,
redesigned, optimized, refined, upgraded, standardized, reengineered

### Delivered / Executed (you shipped / completed)
delivered, executed, shipped, launched, completed, implemented, deployed,
finalized, produced, achieved, attained, fulfilled, realized

### Influenced / Persuaded (people, deals, alignment)
negotiated, secured, closed, won, persuaded, convinced, aligned, influenced,
mobilized, rallied, championed, advocated, brokered, partnered

### Advised / Analyzed (expertise, insight)
advised, analyzed, assessed, evaluated, audited, diagnosed, forecasted,
modeled, researched, quantified, identified, uncovered, recommended

### Built people / teams (leadership of humans)
mentored, coached, hired, recruited, developed, trained, onboarded, retained,
empowered, promoted, cultivated, unified

## Weak openers to remove

Replace these — they describe duties, not results:

- "Responsible for…" → name the result you owned.
- "Worked on / Helped with / Assisted in…" → state what *you* drove.
- "Involved in / Participated in / Part of…" → claim your specific contribution.
- "Tasked with / Duties included / In charge of…" → lead with the outcome.
- "Handled / Dealt with / Took care of…" → use a precise impact verb.

## Before / after

| Weak | Strong |
|------|--------|
| Responsible for managing the sales team | Led a 12-person sales team to 130% of quota for 3 consecutive quarters |
| Worked on improving site performance | Cut median page load 2.4s → 0.8s, lifting conversion 18% |
| Helped with the migration to AWS | Drove a zero-downtime migration of 40 services to AWS, saving $480K/yr |
| Involved in hiring | Recruited and onboarded 22 engineers, reducing time-to-hire from 60 to 28 days |

## Rules of thumb

- One verb per bullet, at the front. Never start with "I", a date, or a noun.
- Match tense to time: current role = present tense; everything else = past tense.
- Don't reuse a verb within the same role; vary across the whole document.
- The verb sets up a *result* — if a bullet has no measurable or concrete outcome,
  see `quantification-guide.md`.
